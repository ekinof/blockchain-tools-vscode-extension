# Change Log

All notable changes to the "crypto-blockchain-tools" extension will be documented in this file.

## [Unreleased]

## [0.0.1] - 2026-02-27

### Added
- Ethereum (ERC20) address generation with EIP-55 checksumming
- Ethereum address checksum validation and case toggling
- Ethereum transaction hash generation and validation
- Bitcoin address generation (P2PKH, P2SH, Bech32) and validation
- Bitcoin transaction hash generation and validation
- Solana address generation and validation (Base58)
- Solana transaction signature generation and validation
- Quick action menu accessible via `Ctrl+Alt+Shift+B` / `Cmd+Alt+Shift+B`
- Settings to customize quote style, 0x prefix, BTC address type, and enabled blockchains
